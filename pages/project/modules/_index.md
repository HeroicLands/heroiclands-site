# Installing Foundry VTT Modules

To install any of the modules above:

1. Find the desired module and copy its **Manifest URL** (click the little copy icon to the right of the URL).
2. In Foundry VTT, go to the "Configuration and Setup" screen, then select the "Add-on Modules" tab.
   ![FVTT Add-on Modules](https://cdn.heroiclands.org/images/fvtt-add-on-modules.webp)
3. Click "Install Module".
4. Paste the manifest URL into the **Manifest URL** text box at the bottom of the dialog and click "Install".
