**HârnMaster 3** is a mature Foundry VTT implementation of Columbia Games' classic skill-based RPG. It is the ancestor of [Song of Heroic Lands](/project/sohl/) and remains fully supported for groups running HârnMaster campaigns.

# System

- [HârnMaster 3 on GitHub](https://github.com/toastygm/HarnMaster-3-FoundryVTT) — source, releases, and installation.

See [Modules](/project/modules/) for general instructions on installing Foundry modules and for system-agnostic location and adventure modules that work with HârnMaster 3.

# HârnMaster-Specific Modules

The modules below extend or modify the HârnMaster 3 system itself. They are only usable with HM3 — **not** with *Song of Heroic Lands*.
