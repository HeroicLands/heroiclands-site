**Song of Heroic Lands** (SoHL) is the Foundry VTT system designed for the world of Thalorna. It is descended from, and largely compatible with, the HârnMaster 3 ruleset, and is the system all new Heroic Lands content is being built against.

SoHL is currently in active development. In the meantime, the links below cover the system's current state — what the rules do, how characters work, and how to try it out.

# Documentation

- [User Guide](/project/sohl-user-guide/) — orientation for GMs and players: concepts, table conventions, system overview.
- [Rules](/project/sohl-rules/) — the full rules reference: attributes, skills, combat, mysticism, healing, and more.
- [Quickstart](/project/sohl-quickstart/) — a compact getting-started path for new tables.
- [Character Creation](/project/sohl-character-creation/) — step-by-step walkthrough for rolling up a new PC.

# System Reference

The catalogs under [Reference](/project/reference/) describe the building blocks SoHL uses to represent characters, creatures, and gear — skills, traits, mystical abilities, domains, armor, weapons, and the rest. Browse them directly if you're designing a character or an NPC and want to know what a given item or ability does.

# Status

SoHL is not yet packaged as a Foundry module; installation instructions will appear on the [Modules](/project/modules/) page once the initial release is available.
