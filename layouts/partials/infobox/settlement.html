{{- /* Settlement infobox. A floated card rendered at the top of a settlement page's main content column (type: doc, category: settlement). Mirrors the polity infobox: a slug-driven profile card whose location/language/pantheon rows resolve to real pages and render as links when a match exists. Rows: - Name — name.full (fallback: title) - Type — settlementType, rendered verbatim (free-text label like "Imperial City (Capital)" or "Market & Temple Town"; NOT humanized, so qualifiers and ampersands survive) - Population — population, rendered verbatim ("~450,000") - Region — the political chain. Climbs parent.polities from the settlement's immediate polity up through its parent polities (nome/province → empire) and renders the chain root→leaf as links joined by " — ", e.g. "The Empire of Ta'Kheperu — The Capital Nome". - Language — languages slug(s) resolved to the language page(s) - Pantheon — pantheons slug(s) resolved to the pantheon page(s) Every section is guarded so missing data silently disappears. */ -}} {{- $ctx := . -}} {{- $name := "" -}}{{- with .Params.name.full }}{{ $name = . }}{{ end -}} {{- if not $name }}{{ $name = .Title }}{{ end -}} {{- $stype := .Params.settlementType -}} {{- $pop := .Params.population -}}{{/* integer; 0 is a valid value (e.g. a necropolis), so guard with `ne nil` rather than `with` */}} {{- $langs := .Params.languages | default slice -}} {{- $panths := .Params.pantheons | default slice -}} {{- $polities := slice -}} {{- with .Params.parent }}{{- with .polities }}{{ $polities = . }}{{ end -}}{{- end -}} {{- /* Build the political chain by climbing parent.polities. Start at the settlement's immediate polity and walk up each polity's own first parent polity. Prepend each page so the collected order is root→leaf. Fixed-depth loop with a guard (real chains are 1–2 deep; 6 is ample) since Hugo has no while loop. */ -}} {{- $chain := slice -}} {{- $cur := "" -}} {{- if gt (len $polities) 0 }}{{ $cur = index $polities 0 }}{{ end -}} {{- range (seq 6) -}} {{- with $cur -}} {{- $pg := first 1 (where $ctx.Site.RegularPages "Params.slug" .) -}} {{- with index $pg 0 -}} {{- $chain = (slice .) | append $chain -}} {{- $cur = "" -}} {{- with .Params.parent }}{{- with .polities -}} {{- if gt (len .) 0 }}{{ $cur = index . 0 }}{{ end -}} {{- end }}{{- end -}} {{- else -}} {{- $cur = "" -}} {{- end -}} {{- end -}} {{- end -}}
